API klíč

Metea získává data z OpenWeatherMap. K načtení dat potřebuje Váš API klíč. Zadejte jej prosím níže.

Co to je API?

API je způsob, pomocí něhož poskytovatel služby (v tomto případě OpenWeatherMap) poskytuje svá data tak, aby mohla být implementována do aplikací (ve strojově čitelném formátu). Data od OpenWeatherMap můžete získat pouze, pokud máte svůj API klíč. Jedná se o identifikátor, kterým se prokazujete serveru, že máte na data nárok. OpenWeatherMap poskytuje jak zpoplatněné služby, tak i služby zdarma, které jsou ale pro běžné použití jednotlivci více než dostatečné. Zaregistrovat se a získat nový API klíč
